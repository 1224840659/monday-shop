## Requirement
1. Laravel 5.5.13

## Installation
通过`git`下载源码或者直接下载压缩包
```shell
git clone / download zip
```
**修改数据库等配置**

在根目录下执行数据库迁移生成表
```shell
php artisan migrate
```
在根目录下执行数据库填充生成数据
```shell
php artisan db:seed

```
